# Cloud Kitchen Management System

A Cloud Kitchen Management System developed using ASP.NET and MSSQL to streamline food ordering, kitchen operations, and order management.

The project helps manage customers, menu items, orders, and administrative operations through a database-driven system with a user-friendly interface.

---

## 🚀 Features

- Customer registration and login
- Food menu management
- Online order placement
- Order tracking and management
- Email Notification
- Admin dashboard
  - Manage Menu
  - Manage Order
  - Manage Area
  - Reports
- Database integration
- CRUD operations
- Responsive interface

---

## 🛠 Technologies Used

- ASP.NET
- C#
- MSSQL Server
- HTML5
- CSS3
- JavaScript
- Bootstrap

## ▶️ How To Run

1. Clone the repository

```bash
git clone https://github.com/vinitsonii/Cloud_Kitchen.git
```

2. Open the project in Visual Studio

3. Configure MSSQL database connection

4. Change Connection String and Mail credentials in config
  
6. Admin : admincloudkitchen2026@gmail.com/password
   
7. Run the project using IIS Express
